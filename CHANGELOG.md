# Changelog

## 1.1.0 - 24 April 2026

- Fixed Android image sharing so supported apps receive the saved image file.
- Added favorites and tags to saved gallery items.
- Added quick filters for all items, favorites, camera, library, and top tags.
- Expanded the detail screen to edit tags and favorite state.
- Added a Settings screen with sign out, theme toggle, local data clear, and release details.
- Added an in-app privacy policy screen and support for an optional hosted privacy policy URL.
- Updated release and store documentation for the new product scope.
